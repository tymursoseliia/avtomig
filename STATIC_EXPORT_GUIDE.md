# 📦 Статический экспорт для Таймвеб (Вариант B)

## Что это?
Создание статической версии сайта (обычные HTML/CSS/JS файлы) для загрузки на обычный веб-хостинг Таймвеб.

## ⚠️ Важно!
При статическом экспорте **НЕ будут работать**:
- ❌ API маршруты (`/api/telegram` - отправка форм в Telegram)
- ❌ Динамические данные из Supabase
- ❌ Админ-панель для добавления автомобилей

✅ **Будет работать**:
- Все страницы сайта (главная, о нас, команда, отзывы, каталог)
- Дизайн и стили
- Навигация между страницами
- Формы (но без отправки в Telegram)

---

## 🚀 Шаг 1: Подготовка проекта

### 1.1. Откройте файл `next.config.js`

Измените его так:

```js
/** @type {import('next').NextConfig} */
const nextConfig = {
  output: 'export',
  images: {
    unoptimized: true,
  },
}

module.exports = nextConfig
```

### 1.2. Создайте файл `.nojekyll` в папке `public`

```bash
touch public/.nojekyll
```

---

## 🛠️ Шаг 2: Сборка статического сайта

В терминале выполните:

```bash
# Установите зависимости (если еще не установлены)
bun install

# Создайте статический экспорт
bun run build
```

Статические файлы будут созданы в папке `out/`

---

## 📤 Шаг 3: Загрузка на Таймвеб

### Через FileZilla или другой FTP-клиент:

1. **Подключитесь к FTP** вашего хостинга Таймвеб
   - Хост: ваш_домен (из панели Таймвеб)
   - Логин: ваш FTP логин
   - Пароль: ваш FTP пароль

2. **Найдите папку `public_html`** или `www` на сервере

3. **Загрузите ВСЁ содержимое папки `out/`** в `public_html/`
   ```
   out/
   ├── index.html
   ├── about.html
   ├── team.html
   ├── reviews.html
   ├── catalog.html
   ├── _next/
   └── ...
   ```

4. **Подождите** завершения загрузки всех файлов

---

## ✅ Шаг 4: Проверка

Откройте ваш сайт: `http://ваш-домен.ru`

Сайт должен выглядеть правильно со всеми стилями!

---

## 🔧 Настройка .htaccess (для красивых URL)

Создайте файл `.htaccess` в `public_html/`:

```apache
# Включить перезапись URL
RewriteEngine On

# Перенаправить на HTTPS (опционально)
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

# Убрать .html из URL
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteCond %{REQUEST_FILENAME}.html -f
RewriteRule ^(.+)$ $1.html [L,QSA]

# Главная страница
DirectoryIndex index.html
</apache>
```

## ❗ Ограничения статической версии

### Формы не будут работать
Формы заявок не смогут отправлять данные в Telegram.

**Решение:** Замените формы на:
1. Ссылку на Telegram: `https://t.me/OreonAuto`
2. Email форму через PHP
3. Виджет Telegram

### Каталог автомобилей будет пустым
Нет подключения к Supabase базе данных.

**Решение:**
- Вручную добавьте HTML с автомобилями
- Или используйте Node.js хостинг (Вариант C)

---

## 💡 Рекомендация

Если вам нужны рабочие формы и база данных - используйте **Вариант C (Node.js хостинг)** или **Netlify** (бесплатно и уже настроено).

Статический экспорт подходит только для **презентационного сайта** без динамических функций.
